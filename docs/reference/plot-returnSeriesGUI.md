# Return series plots

A graphical user interface to display finanical time series plots.
`returnSeriesGUI` opens a GUI for return series plots.

## Usage

``` r
returnSeriesGUI(x)
```

## Arguments

- x:

  an object of class `"timeSeries"` or any other object which can be
  transformed by the function `as.timeSeries` into an object of class
  `"timeSeries"`.

## Value

`NULL`, invisibly. `returnSeriesGUI` is used for the graphical user
interface it provides.
