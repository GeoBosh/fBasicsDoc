# Generalized Hyperbolic Distribution Slider

Displays interactively the dependence of the generalized hyperbolic
distribution on its parameters.

## Usage

``` r
ghSlider()
```

## Value

a tcl/tk based graphical user interface.

This is a nice display for educational purposes to investigate the
densities and probabilities of the generalized hyperbolic distribution.
