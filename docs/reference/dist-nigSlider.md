# nigerbolic Distribution Slider

Displays interactively the dependence of the nigerbolic distribution on
its parameters.

## Usage

``` r
nigSlider()
```

## Value

a tcl/tk based graphical user interface.

This is a nice display for educational purposes to investigate the
densities and probabilities of the invetrse Gaussian distribution.
