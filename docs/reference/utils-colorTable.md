# Table of colors

Displays a table of color codes and plots the colors themselves.

## Usage

``` r
colorTable(cex = 0.7)
```

## Arguments

- cex:

  a numeric value, determines the character size in the color plot, the
  default is 0.7.

## Value

a table of plot colors with the associated color numbers

## See also

[`characterTable`](https://geobosh.github.io/fBasicsDoc/reference/utils-characterTable.md),
[`symbolTable`](https://geobosh.github.io/fBasicsDoc/reference/utils-symbolTable.md)

## Examples

``` r
colorTable()
```
