# Generalized Lambda Distribution Mode

Computes the mode of the generalized lambda distribution.

## Usage

``` r
gldMode(lambda1 = 0, lambda2 = -1, lambda3 = -1/8, lambda4 = -1/8)
```

## Arguments

- lambda1:

  location parameter.

- lambda2:

  scale parameter.

- lambda3:

  first shape parameter.

- lambda4:

  second shape parameter.
